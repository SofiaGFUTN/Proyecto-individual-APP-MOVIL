# Proyecto-individual-APP-MOVIL

Conversor de monedas
Una aplicación capas de indicar el cambio de la moneda a elección del usuario

Funciones:
-Poder saber el cambio de la moneda (Ej. USD - > Colones)

-Una barra de selección con los distintos cambios de monedas

-Poder escanear números desde la cámara para una conversión más fácil

-Tomar imágenes de galería para también tener una conversión fácil
